Persistent Identifier

long-lasting digital name*

An ORCID iD is a PID for researchers that records professional activities and disambiguates one researcher from another.

(Werf 2022)

3

PIDs: Publication level

Publication level can be the publication, or sections of a publication: chapters for book, time stamps for video, or cordinates for a 3D model, etc. A persistent identifier for a publication is a warrant that the publication both can be found and retrieved as well as distinguished from other similar publications.***

DOI is an example. A Digital Object Identifier (DOI) is an alphanumeric string assigned to uniquely identify an object. It is tied to a metadata description of the object as well as to a digital location, such as a URL, where all the details about the object are accessible.

Open Data: licencing as Creative Commons Zero (CC0)

Data that is openly accessible and usable to others* Open data is data that is openly accessible, exploitable, editable and shared by anyone for any purpose, even commercially. Open data is licensed under an open license. The default licence for data is CCO which is intended to be copyright free.

RADAR4Culture helps you publish your research data on tangible and intangible cultural assets as easily as possible - without neglecting convenience, data security or legal aspects.

(McMurry et al. 2017)

9

Publication metadata: accessible, open standards, machine readable

Data about data that is freely accessible on the internet and is ouside of copyright or is openly licenced for reuse. Access to the metdata is enhanced by open standards and use of identifiers identifiers.

The FAIR Principles go into detail on the topic of Meta(data) as the how of making metadata accessible varies according to circumstances and guidance is more important for specific use cases to take steps of providing accessibility. See the FAIR Principles.

(‘FAIR Principles’ n.d.)
